Featured ProJects

DiceForge

Dice Forge was developed as a collaborative team project to deliver a minimalist, beginner-friendly toolkit for new DnD players. Serving as the UI Developer on the application sub-team, I built the entire user interface and core navigation skeleton from scratch. Additionally, I engineered the app's interactive Dice Roller module, implementing responsive layouts, medieval aesthetics with parchment surfaces and custom gold frames, and multi-threaded coroutine animations that sync tactile audio feedback directly with physical dice spin states.

Kotlin • Jetpack Compose • SQLite • Android Studio

Tale Trail

An immersive, native Android storytelling application built using Kotlin. Trail Tail features personalized child-naming systems, dynamic identity paths, and interactive branching storylines. Designed with custom ConstraintLayout engines and immersive full-screen UX components, it delivers a seamless, adaptive digital storybook environment across different device screens.
